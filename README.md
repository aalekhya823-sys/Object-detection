# YOLOv8 Object Detection (Image & Video)

## Overview

This project uses YOLOv8 to detect objects in images and videos with bounding boxes and confidence scores.

## Requirements

* Python
* OpenCV
* Ultralytics YOLOv8

Install:

```
pip install ultralytics opencv-python
```

## Features

* Image Object Detection
* Video Object Detection
* Real-time Detection
* Confidence Score

## Model

* YOLOv8 (yolov8n.pt)

## Output

* Bounding boxes
* Object labels
* Confidence percentage
